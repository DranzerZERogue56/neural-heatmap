Grid
Grid Size
The number of neurons per row and column (N×N). Larger grids show finer detail but cost more to compute.
60
Neighbor Mode
4-dir spreads activation up/down/left/right. 8-dir also includes diagonals, producing rounder, more organic patterns.
Neuron Behavior
Fire Threshold
A neuron fires when its activation exceeds this value. Lower values make neurons trigger more easily, creating denser patterns.
0.40
Propagation Strength
How much activation a firing neuron spreads to its neighbors. Higher values create waves that travel further across the grid.
0.30
Decay Rate
How quickly a neuron's activation fades when it isn't being stimulated. Higher values make patterns dissolve faster.
0.05
Refractory Period
How many ticks a neuron must cool down after firing before it can fire again. Creates rhythmic, wave-like behavior.
5
Stimulation
Seed Rate
Number of random neurons stimulated each tick. More seeds produce busier, more chaotic patterns.
5
Seed Intensity
How much activation is injected into each randomly stimulated neuron. Higher values make seeds more likely to trigger a firing cascade.
0.60
Rendering
Glow Intensity
Controls the bloom/glow effect on highly activated neurons. Higher values create a stronger neon glow.
0.60

Simulation
Tick Speed
How many simulation ticks run per second. Faster speeds make patterns evolve more quickly.
